RestFul Routes

Action URL HTTP Verb Description
index /pokemons GET Display all the pokemons
new /pokemons/new GET Render a form to create new pokemon
create /pokemons POST Add new pokemon in the dabatase
show /pokemons/:id GET Display all the data related to one pokemon
edit /pokemons/:id/edit GET Render a form containing the data related to one pokemon, user can modify the data in form
update /pokemons/:id PUT Update the data related to one pokemon in the database
delete /pokemons/:id DELETE Delete one pokemon in the database
